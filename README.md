🎮 Игра "Угадай слово" (Виселица)
📌 Описание

Это простая игра на Python, где нужно угадать слово по буквам.
Компьютер загадывает случайное слово из списка, а игрок пытается его отгадать.

У игрока есть ограниченное количество попыток (жизней).
Если буква угадана — она открывается в слове.
Если нет — уменьшается количество жизней.

⚙️ Как работает программа

Список слов

words = ["яблоко", "машина", "программа", "ТалантАгай", "Жантай"]

Здесь задаются слова, из которых случайно выбирается одно.

Выбор случайного слова

word = random.choice(words)

Программа случайно выбирает слово из списка.

Переменные

guessed = []
lives = 6
guessed — список уже введённых букв
lives — количество попыток

Основной цикл

while lives > 0:

Игра продолжается, пока не закончатся жизни.

Отображение слова

for letter in word:
    if letter in guessed:
        display += letter + " "
    else:
        display += "_ "

Показывает:

угаданные буквы
и _ вместо неизвестных

Ввод буквы

guess = input("Введи букву: ").lower()

Игрок вводит букву, она переводится в нижний регистр.

Проверка
Если буква уже вводилась — выводится сообщение
Если буквы нет в слове — уменьшается lives
Если есть — открывается в слове

Победа

if "_" not in display:

Если нет _, значит слово угадано.

Поражение
Если жизни закончились:

if lives == 0:

программа показывает правильное слово.
