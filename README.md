 Общее описание

Этот код — простая текстовая игра «Угадай слово» (аналог «Виселицы»).
Программа случайным образом выбирает слово, а игрок пытается угадать его по буквам за ограниченное количество попыток.

 Как работает программа
1. Импорт библиотеки
import random

Нужен модуль random, чтобы случайно выбрать слово.

2. Список слов
words = ["яблоко", "машина", "программа", "ТалантАгай", "Жантай"]

Список возможных слов, из которых программа выберет одно.

3. Выбор случайного слова
word = random.choice(words)

Компьютер случайно выбирает одно слово из списка.

4. Переменные игры
guessed = []
lives = 6
guessed — список уже введённых букв
lives — количество попыток (жизней)
5. Приветствие
print("Игра: Угадай слово!")
print("Слово состоит из", len(word), "букв")

Показывает игроку, сколько букв в слове.

6. Основной игровой цикл
while lives > 0:

Цикл работает, пока есть жизни.

7. Отображение слова
display = ""

for letter in word:
    if letter in guessed:
        display += letter + " "
    else:
        display += "_ "
Показывает угаданные буквы
Неугаданные заменяет на _
8. Проверка победы
if "_" not in display:
    print("Ты выиграл!")
    break

Если все буквы открыты — игрок выиграл.

9. Ввод буквы
guess = input("Введи букву: ").lower()

Игрок вводит букву (в нижнем регистре).

10. Проверка на повтор
if guess in guessed:
    print("Ты уже вводил эту букву")
    continue

Если буква уже была — предупреждение.

11. Добавление буквы
guessed.append(guess)

Сохраняем введённую букву.

12. Проверка угадывания
if guess not in word:
    lives -= 1
    print("Неправильно! Осталось жизней:", lives)
else:
    print("Есть такая буква!")
Если буквы нет — минус жизнь
Если есть — сообщаем об успехе
13. Проигрыш
if lives == 0:
    print("Ты проиграл. Слово было:", word)

Если закончились жизни — игрок проиграл.
